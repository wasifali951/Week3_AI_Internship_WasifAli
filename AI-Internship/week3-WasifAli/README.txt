===========================================
   Week 3 – AI Internship @ Code Saviours
     Machine Learning with Python
===========================================

This folder contains implementations of data preprocessing, 
machine learning algorithms, model evaluation, and a 
Student Performance Prediction project developed during Week 3.

-------------------------------------------
1. Data Preprocessing
-------------------------------------------
File: preprocessing.py

Description:
- Handles missing values (mean, median, mode).
- Encodes categorical variables (label encoding, one-hot encoding).
- Normalizes and standardizes numerical features.
- Saves cleaned dataset for ML models.

How to Run:
python preprocessing.py

-------------------------------------------
2. Linear Regression
-------------------------------------------
File: linear_regression.py

Description:
- Implements Linear Regression for predicting continuous values.
- Compares predicted vs actual results using regression metrics.

How to Run:
python linear_regression.py

-------------------------------------------
3. Logistic Regression
-------------------------------------------
File: logistic_regression.py

Description:
- Implements Logistic Regression for binary classification tasks.
- Evaluates performance with classification metrics.

How to Run:
python logistic_regression.py

-------------------------------------------
4. K-Nearest Neighbour (KNN)
-------------------------------------------
File: knn.py

Description:
- Implements KNN algorithm for classification.
- Predicts labels based on nearest data points.

How to Run:
python knn.py

-------------------------------------------
5. Decision Tree
-------------------------------------------
File: decision_tree.py

Description:
- Implements Decision Tree for both classification and regression.
- Visualizes the decision boundaries.

How to Run:
python decision_tree.py

-------------------------------------------
6. Random Forest
-------------------------------------------
File: random_forest.py

Description:
- Implements Random Forest (ensemble learning).
- Compares accuracy and feature importance.

How to Run:
python random_forest.py

-------------------------------------------
7. Model Evaluation – Classification
-------------------------------------------
File: evaluation_classification.py

Description:
- Evaluates classification models with:
  Confusion Matrix, Accuracy, Precision, Recall, F1-score.

How to Run:
python evaluation_classification.py

-------------------------------------------
8. Model Evaluation – Regression
-------------------------------------------
File: evaluation_regression.py

Description:
- Evaluates regression models with:
  R² Score and Mean Squared Error (MSE).

How to Run:
python evaluation_regression.py

-------------------------------------------
9. Student Performance Prediction
-------------------------------------------
File: student_performance_model.py

Description:
- Main project for Week 3.
- Predicts student performance using Linear Regression 
  and Random Forest Regression.
- Compares model accuracy and visualizes results.

How to Run:
python student_performance_model.py
